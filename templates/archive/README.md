### page--front--safety-original.html.twig
- **Original Purpose**: Front page template focused on AI-Powered Safety Monitoring for Philadelphia
- **Archived Date**: February 5, 2026
- **Reason**: Replaced with community-focused home page template
- **Features**:
  - Animated header with "Forseti Life" branding
  - Safety-focused messaging: "AI-Powered Safety Monitoring for Philadelphia"
  - Call-to-action buttons for Safety Map, Mobile App, and About page
  - Designed for AmISafe crime monitoring context

## Restoration

To restore this template:
1. Copy from this archive directory to theme templates directory
2. Place at: `web/themes/custom/forseti/templates/page/page--front.html.twig`
3. Clear Drupal cache: `drush cr`
